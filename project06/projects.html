This page highlights specific projects I've done while at

IST 387 – Energy Forecasting with SVM Models

We were assigned a dataset from an energy company in South Carolina focused on energy usage and temperatures during July. Using R and an SVM model, I simulated a +5°F scenario where 90% of days exceeded grid capacity. I recommended infrastructure upgrades and dynamic pricing to help mitigate risk. This was my first full data science project with machine learning.

IST 356 – Sports Betting Odds API

I built a Python pipeline that pulls JSON data from a sports odds API and displays it in a Streamlit app. The project compares lines from FanDuel, DraftKings, and other bookmakers to show which site offers the best odds. It taught me how to handle APIs, clean data, and build dashboards for real users.

IST 421 – LinkedIn Job Analysis

I scraped 30K+ job postings from LinkedIn, cleaned them in R, and visualized patterns in salary and location. Remote jobs had higher average pay, and urban regions had more opportunities. I presented this at Poster Day and honed both my data storytelling and public speaking skills.
